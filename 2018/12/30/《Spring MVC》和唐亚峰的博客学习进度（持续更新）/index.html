Spring MVC和Spring Boot学习进展记录

2018-12-30

1.《看透Spring MVC》

  • 目前看完了HandlerMapping部分，果断时间继续看HandlerAdapter部分，最近几天就不要再看了

  • 博客部分写到DispatcherServlet了暂时这几天就不写了，反正我对Spring MVC也比较熟悉了，看完自信了很多

  • 之后有时间再慢慢更新博客，就好了。要让博客跟上自己看的内容，而不是一味地追求进度，看了后面忘记前面，要用写博客这个事情，加深理解，并且复习前面看过的内容。

2.《Spring Boot 唐亚峰》（基于Spring Boot2.0）

  • 核心知识点

    1. 如何根据模板新建一个Spring Boot项目、Spring Boot的jar包和插件、约定优于配置的编程理念
    2. 配置文件（多环境）以及@ConfigurationProperties、@PropertySource 等注解的用法
    3. 日志的简单介绍
    4. thymeleaf模板的整合：基本使用和一些细节(测试的时候不要设置带缓存),看的还行
    5. JdbcTemplate操作数据库
    6. 整合Spring Data JPA 约定优于配置的最明显体现
    7. 整合MyBatis，讲的基本清楚，因为例子比较简单，可以说只是入门；需要再努力细致地看下关联查询和翻页插件
    8. 整合Redis、 Spring Cache
    9. 消息队列，消息队列的延迟队列应用，讲述的还不够深入

    关于关联查询和分页的内容，看《MyBatis从入门到精通》，认真读完，应该就理解得差不多了

    1. actuator 服务监控与管理，自定义端点

    2. Spring Boot Admin监控客户端端点，非常方便

    3. 定时任务

    4. 文件上传

    5. 全局异常 主要是指处理controller层的异常，@RestControllerAdvice 和@ExceptionHandler注解

    6. 数据验证的相关注解

    7. 搭建websocket在线聊天

    8. shiro安全框架

  • 目前，上述教程的文章都已经看完，讲的不够详细，但是对整个知识体系有了整体的了解，所以该教程的学习目的已经达到
